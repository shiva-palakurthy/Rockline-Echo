ğŸ§  How Mood Simulation Works
Due to browser limitations, all moods use the same base voice.
Real emotional or gender-based voices are not available natively.
To simulate moods, Rockline Echo modifies:
Pitch (voice height)
Rate (speech speed)
Speech patterns (pauses, prefixes, emphasis)
This creates a perceived emotional difference while remaining fully offline and browser-only.
